# Star Rating Widget using HTML & CSS | CodingNepal

Um projeto simples de **sistema de avaliação com estrelas (star rating)** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **rating widget com estrelas**, ideal para avaliações de produtos, serviços ou conteúdo.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Sistema de avaliação com estrelas
- Interação com hover
- Design simples e moderno
- Sem uso de JavaScript

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
